# MAP Estimate Best Practices

Extended MAP guidance (funding tiers, environment sizing, data transfer rules, QA checklists) is available in the private steering repository.

## Quick Reference

- **On-Demand pricing** as baseline — MAP ARR uses list price
- **Include all environments** — Production, Staging (~20%), Development (~20%), Shared Services
- **Group naming**: `Production`, `Staging`, `Development`, `Shared Services`, `Disaster Recovery`

## Access Private Steering

If you have access to the private steering repo, the full MAP best practices and field quality rules are available there. If not, request access from the repo owner.
